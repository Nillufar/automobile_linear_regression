## Used Car Price Analysis
This project analyzes used car prices.
It includes data cleaning, visualizations, and a linear regression model to understand what factors influence car prices.
## Project Contents
analysis.ipynb – full Jupyter Notebook analysis
main.py – optional Python script version
autos.csv – dataset (add your own)
outputs/ – plots, cleaned data, and model results
## How to Use
Place autos.csv on your Desktop or inside this project.
Update the path if needed:
DATA_PATH = "/Users/nilufarkurbonova/Desktop/autos.csv"
Run the project:
Jupyter Notebook
jupyter notebook analysis.ipynb
Python Script
python3 main.py
## What You Get
All results are saved in outputs/:
Cleaned dataset
Histograms & boxplots
Scatter plots
Correlation heatmap
Linear regression metrics
Depreciation forecast plot
## Tools Used
Python, Pandas, NumPy, Matplotlib, Scikit-learn, Jupyter Notebook.
