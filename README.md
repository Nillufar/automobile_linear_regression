# Linear Regression
This project analyzes used car prices using autos.csv. It performs data cleaning, exploratory data analysis (EDA), visualizations, and a simple linear regression model to understand price drivers and predict depreciation.

## How to Use
Place autos.csv on your Desktop or inside this repository.
Update the path in the script/notebook if needed:
DATA_PATH = "/Users/nilufarkurbonova/Desktop/autos.csv"
Run the analysis:
Jupyter Notebook:
jupyter notebook analysis.ipynb
Python script:
python3 main.py
What the Code Produces
All outputs are saved in the outputs/ folder:
Cleaned dataset
Histograms
Category bar charts
Correlation heatmap
Boxplots
Scatter plots
Linear regression metrics
Depreciation forecast plot
Summary text file
Tools
Python, Pandas, NumPy, Matplotlib, Scikit-learn, Jupyter Notebook.
